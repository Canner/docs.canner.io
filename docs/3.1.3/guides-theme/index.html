Theme

Introduction

Currently our CMS components are based on an awesome React UI library called Antd. Customizing UI themes are a frequent request at Canner, by passing theme settings in canner.cloud.js.

  • Antd variable list

For example, if you want to replace primary-color with #07a4b8, you can add the cloud.config.js below in your working directory.

canner.cloud.js

module.exports = {
  "theme": {
    "primary-color": "#07a4b8"
  }
}

Canner Theme
