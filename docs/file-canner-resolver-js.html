canner.resolver.js

canner.resolver.js should export a resolver object, each key must match with your data's key.

Learn more about Resolvers

canner.resolver.js

// ...
export default {
  posts: postsResolver,
  info: infoResolver
}

If you don't need any resolvers, you can just exports a empty object.

canner.resolver.js

export default {}
